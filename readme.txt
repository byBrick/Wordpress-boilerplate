=== byBrick WordPress boilerplate ===

== Description ==

A WordPress HTML5 boilerplate for new WordPress projects. Suitable for most projects. 

== Features ==

* Loads jQuery from Google CDN
* Loads Selectivizr from cdnjs (for Internet Explorer only)
* Loads html5shiv from Google CDN (for Internet Explorer only)
* Adds browser type to body class
* Custom login image
* Removes random junk from wp_head
* Adds Google Analytics code automatically, change UA-XXXXX-X in the functions.php file
* Out of the box post thumbnails and menu support
* CSS Reset & Clearfix
* Default typography
* Default WordPress CSS classes
